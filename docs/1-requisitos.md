# 1. Tecnologías y requisitos previos

## Backend, desarrollado con Laravel

**Requisitos**:

- **PostgreSQL** o **MySQL**/**MariaDB**
- Composer >= 2.8
- PHP >= 8.2
- Laravel >= 12

<img src="./assets/laravel.svg" width="120" alt="Laravel Logo">

### Storage

Las imágenes utilizadas en la base de datos (ejercicios, entrenamientos y perfiles de usuarios de prueba) **no están incluidas en el repositorio**, más concretamente en el directorio `api/storage/app` debido a posibles restricciones de **copyright**.
