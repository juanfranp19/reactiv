# Reactiv

Aplicación para gestionar rutinas y entrenamientos, controlar accesos y salidas, administrar usuarios, socios, entrenadores, entre otras muchas cosas dependiendo del rol: socio, entrenador o administrador. Además cuenta con una web aparte para simular los accesos y salidas del gimnasio.

### Estructura de directorios

```
reactiv
│
├── /acceso     # Web de acceso al gimnasio
├── /api        # Backend
├── /docs       # Archivos de documentación y assets
├── /web        # Web principal de la aplicación
├── LICENSE
└── README.md
```
